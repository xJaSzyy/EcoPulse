# EcoPulse

**EcoPulse** — система мониторинга экологических данных с веб-интерфейсом и backend API. Проект состоит из двух основных сервисов: backend на .NET 9.0 и frontend на Vue.js, с PostgreSQL 18 в качестве базы данных.

## Описание сервисов

| Сервис | Технологии | Описание |
|--------|------------|----------|
| **EcoPulseBackend** | .NET 9.0, ASP.NET Core, EF Core | REST API для обработки экологических данных, геопространственных запросов и аналитики. Поддерживает CRUD операции, авторизацию JWT и интеграцию с внешними источниками данных. |
| **EcoPulseFrontend** | Vue.js 3, OpenLayers, TypeScript | Современный веб-интерфейс с интерактивными картами, дашбордами и визуализацией данных. Использует Vue Router и Pinia для управления состоянием. |
| **PostgreSQL** | Postgres:18 | База данных для хранения геоданных, временных рядов и метаданных. Поддерживает PostGIS для пространственных запросов. |
